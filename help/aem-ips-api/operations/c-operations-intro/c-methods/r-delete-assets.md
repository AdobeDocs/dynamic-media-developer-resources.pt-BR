## Exemplos {#section-aaad1933bf86479eb6cb476cec7d4587}

Esta amostra de código envia um identificador para uma empresa e uma matriz de identificadores de ativos em uma `deleteAssetsParam` solicitação para o servidor de serviços da Web. `deleteAssetsReturn` retorna uma contagem de sucesso de 2, indicando que ambos os ativos foram excluídos.

**Solicitação**

```java
<deleteAssetsParam xmlns="http://www.scene7.com/IpsApi/xsd/2008-01-15">
   <companyHandle>c|6</companyHandle>
   <assetHandleArray>
      <items>a|942|1|579</items>
      <items>a|943|1|580</items>
   </assetHandleArray>
</deleteAssetsParam>
```

**Resposta**

```java
<deleteAssetsReturn xmlns="http://www.scene7.com/IpsApi/xsd/2008-01-15">
   <successCount>2</successCount>
   <warningCount>0</warningCount>
   <errorCount>0</errorCount>
</deleteAssetsReturn>
```
