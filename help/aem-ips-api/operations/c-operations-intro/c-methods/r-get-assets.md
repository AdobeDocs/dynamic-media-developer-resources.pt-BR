# getAssets{#getassets}

Retorna ativos do IPS.

Sintaxe

## Tipos de usuário autorizados {#section-4673c1c9f4314160af8b165eb2dd20cc}

* `IpsUser`
* `IpsAdmin`
* `IpsCompanyAdmin`
* `TrialSiteAdmin`
* `TrialSiteUser`
* `ImagePortalAdmin`
* `ImagePortalUser`
* `ImagePortalContrib`
* `ImagePortalContribUser`

>[!NOTE]
>
>Retorna somente os ativos aos quais o usuário tem acesso.
