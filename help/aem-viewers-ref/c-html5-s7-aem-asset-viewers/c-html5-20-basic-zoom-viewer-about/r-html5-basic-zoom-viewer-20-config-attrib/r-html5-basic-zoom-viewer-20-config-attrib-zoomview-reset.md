# ZoomView.reset{#zoomview-reset}

`[ZoomView.|<containerId>_zoomView.]reset=0|1`

<table id="table_49FFD1BC53B846F09A6D214BC8C5C3FE"> 
 <tbody> 
  <tr> 
   <td colname="col1"> <p> <span class="codeph"> 0|1</span> </p> </td> 
   <td colname="col2"> <p> Redefine a porta de visualização quando o quadro (imagem) muda. Se definido como <span class="varname"> 0</span> preserva a porta de visualização atual com o melhor ajuste possível, preservando a proporção da imagem recém-definida. </p> </td> 
  </tr> 
 </tbody> 
</table>

## Propriedades {#section-50bcd15223174bb79ce08b31ea03d682}

Opcional.

## Padrão {#section-7564169749ff4a4996049ea1148cb2a5}

`1`

## Exemplo {#section-96e69b70365f461dae4399e49044ea2f}

`reset=0`
