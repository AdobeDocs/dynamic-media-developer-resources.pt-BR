## Propriedades {#section-f42369774e2740dcb399626a0e4e930e}

Obrigatório. (A menos que o parâmetro `asset` seja usado. Consulte [Suporte a Vídeo Externo](../../../c-html5-s7-aem-asset-viewers/c-html5-video-reference/r-html5-video-viewer-20-external-video-support.md#concept-22c67fee43274a29b28ee16770b1b1f3).)

## Padrão {#section-d016470e92a74f98a18c4ab3489410a5}

Nenhum.

```
video=https://s7d9.scene7.com/is/content/Scene7SharedAssets/Glacier_Climber_MP4
```
