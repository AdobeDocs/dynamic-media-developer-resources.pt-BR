# Notas de versão do Scene7 Publishing System 6.0{#scene-publishing-system-release-notes}

Notas de versão que descrevem os recursos, aprimoramentos e correções do Adobe Scene7 Publishing System 6.0-parte da Adobe Marketing Cloud.

## Novos recursos e melhorias {#section-4bbd330df5a043f7a56eee619ef7c442}

**Geral**

* Language support for Chinese.
* Melhorias administrativas, incluindo:

   * Export list of company users.
   * Envie emails Cco para usuários.
   * Redefina as senhas do usuário para uma senha aleatória que o usuário possa redefinir.

* Aprimorada a utilização da visualização do modelo. Consulte o seguinte para obter mais informações:

**Visualizadores**

* Adição de suporte de localização para texto de dica de ferramenta nas Predefinições do visualizador do Flash AS3 e HTML5.
* Estenda o alcance dos vídeos aos mercados globais adicionando legendas a ativos únicos de eVideo.
* Adição de &quot;Código incorporado&quot; para simplificar a implantação de visualizadores.

   Usar o [!DNL Embed Code] permite revisar o código do visualizador para o tipo de ativo selecionado. Você também pode copiar o código para a área de transferência, de modo que possa colá-lo em suas páginas da Web para a implantação do visualizador. A edição do código não é permitida na caixa de diálogo Incorporar código. .

   The [!DNL Embed Code] feature is available when you open the [!DNL Preview: Viewer List] page or the [!DNL URLs] panel in [!DNL Detail View] for any of the following asset types:

   * Universal Viewers
   * Zoom
   * Catálogos eletrônicos
   * Conjuntos de imagens
   * Swatch Sets
   * Spin
   * Recuos de vídeo
   * Conjuntos de vídeos adaptáveis
   * Video
   * Conjuntos de mídia mista

* Esquemas de metadados

   Os esquemas de metadados definem conjuntos de propriedades de metadados. Cada propriedade do esquema Metadados tem um tipo de dados básico como String, Float, Integer, Date ou Yes/Não. Também é possível definir listas, opções abertas ou opções fechadas. Os esquemas de metadados podem ser usados em todos os fluxos de trabalho de metadados, como pesquisa, edição, importação, exibições de metadados e predefinições de metadados. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS259993e42159a215-1c6a66df1265272619e-7fec.html#WSd968ca97bf03cf72-5e3dd3a113268dc80f5-8000). -->

   Os esquemas de metadados são a tecnologia sucessora dos UDFs e destinam-se a fluxos de trabalho de metadados personalizados, especialmente no contexto do Servidor de Metadados.

* Servidor de metadados

   >[!NOTE]
   >
   >Metadata Server will be available starting in May. Ele deve ser habilitado pelo suporte. Atualmente, o Servidor de Metadados está disponível somente na América do Norte e na Europa.

* The metadata Server exposes a subset of all metadata fields of an asset to a public-facing server and can be requested here by eCom pages or servers. Esses metadados expostos podem ser um conjunto misto de Propriedades de ativos, UDFs, XMP e propriedades de Esquema de metadados. The Metadata Server offers a WEB-API with a variety of search capabilities like searches for metadata, or image similiarity, or both. For each returned asset a list of metadata properties is requested. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS1ffbb36e209a6fc9-44279087131d3ad5622-8000.html). -->
* A importação de metadados pode manipular XMP propriedades

   A importação de metadados é estendida para lidar com propriedades e propriedades XMP, definidas nos esquemas de Metadados.
* Exibição de metadados para exibir somente campos contendo valores

   There is a new predefined Metadata View called &quot;All Properties with Values&quot; that only shows meta properties that contain values. As propriedades (de XMP, Campos definidos pelo usuário ou os novos Esquemas de metadados) sem valores são ocultas e não serão exibidas.
* As configurações de metadados agora são agrupadas na interface de administração do usuário.

   In **[!UICONTROL Setup > Application Setup]**, metadata-related setting pages are now grouped in a top-level section called [!DNL Metadata]. Here you find the setting pages for Metadata View, Metadata Presets, Metadata Schema and the User-Defined Fields.

This desktop application lets you import existing 3D models and author them as 2D vignette files. The resulting vignette files can be uploaded to SPS, where colors and textures can be replaced on-demand.
