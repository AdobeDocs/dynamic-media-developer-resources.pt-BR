# Notas de versão do Scene7 Publishing System 6.0{#scene-publishing-system-release-notes}

Notas de versão que descrevem os recursos, aprimoramentos e correções do Adobe Scene7 Publishing System 6.0-parte da Adobe Marketing Cloud.

## Novos recursos e aprimoramentos {#section-4bbd330df5a043f7a56eee619ef7c442}

**Geral**

* Suporte de idioma para chinês.
* Melhorias administrativas, incluindo:

   * Exportar lista de usuários da empresa.
   * Envie emails Cco para usuários.
   * Redefina as senhas do usuário para uma senha aleatória que o usuário possa redefinir.

   * Zoom
   * Catálogos eletrônicos
   * Conjuntos de imagens
   * Conjuntos de amostras
   * Rotação

   * Conjuntos de vídeos adaptáveis
   * Vídeo

* Exportar arquivos grandes sem ter tempo limite.

* Exportar lista de usuários com grupos e ativos/permissões (administrador).

* Esquemas de metadados

   Os esquemas de metadados definem conjuntos de propriedades de metadados. Cada propriedade do esquema Metadados tem um tipo de dados básico como String, Float, Integer, Date ou Yes/Não. Também é possível definir listas, opções abertas ou opções fechadas. Os esquemas de metadados podem ser usados em todos os fluxos de trabalho de metadados, como pesquisa, edição, importação, exibições de metadados e predefinições de metadados. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS259993e42159a215-1c6a66df1265272619e-7fec.html#WSd968ca97bf03cf72-5e3dd3a113268dc80f5-8000). -->

   Os esquemas de metadados são a tecnologia sucessora dos UDFs e destinam-se a fluxos de trabalho de metadados personalizados, especialmente no contexto do Servidor de Metadados.

* Servidor de metadados

   >[!NOTE]
   >
   >O Servidor de metadados estará disponível a partir de maio. Ele deve ser habilitado pelo suporte. Atualmente, o Servidor de Metadados está disponível somente na América do Norte e na Europa.

* O Servidor de metadados expõe um subconjunto de todos os campos de metadados de um ativo para um servidor aberto ao público e pode ser solicitado aqui por páginas ou servidores do eCom. Esses metadados expostos podem ser um conjunto misto de Propriedades de ativos, UDFs, XMP e propriedades de Esquema de metadados. O Servidor de Metadados oferece uma API da WEB com vários recursos de pesquisa, como pesquisas por metadados, similaridade de imagem, ou ambos. Para cada ativo retornado, uma lista de propriedades de metadados é solicitada. <!-- [More information](http://help.adobe.com/en_US/scene7/using/WS1ffbb36e209a6fc9-44279087131d3ad5622-8000.html). -->
* A importação de metadados pode manipular XMP propriedades

   A importação de metadados é estendida para lidar com propriedades e propriedades XMP, definidas nos esquemas de Metadados.
* Exibição de metadados para exibir somente campos contendo valores

   Há uma nova Exibição de metadados predefinida chamada &quot;Todas as propriedades com valores&quot; que mostra apenas as meta propriedades que contêm valores. As propriedades (de XMP, Campos definidos pelo usuário ou os novos Esquemas de metadados) sem valores são ocultas e não serão exibidas.
* As configurações de metadados agora são agrupadas na interface de administração do usuário.

   Em **[!UICONTROL Setup > Application Setup]**, as páginas de configuração relacionadas aos metadados agora são agrupadas em uma seção de nível superior chamada [!DNL Metadata]. Aqui você encontra as páginas de configuração para Exibição de metadados, Predefinições de metadados, Esquema de metadados e Campos definidos pelo usuário.

Esse aplicativo de desktop permite importar modelos 3D existentes e criá-los como arquivos de vinheta 2D. Os arquivos de vinheta resultantes podem ser carregados no SPS, onde as cores e as texturas podem ser substituídas sob demanda.

## Correções de software {#section-4219d5dd0a024cd3a95aaac45598363c}

Os seguintes problemas foram corrigidos no Servidor de Metadados:

* Solução de um problema que causava a falha de uploads de arquivos EPS muito grandes.
* &quot;Pesquisas de metadados&quot; com lista de resultados longa falham silenciosamente. Esse problema faz com que resultados de pesquisa incorretos sejam exibidos.
