# TotalStreamBitRate{#totalstreambitrate}

Taxa total de bits do fluxo. Valor longo que representa a taxa total de bits do fluxo para o ativo de vídeo.

Usado pelas solicitações `req=set` e `req=mbrSet`para exportar dados de taxa de bits.

Aplica-se somente ao catálogo de conteúdo estático.

## Propriedades {#section-76731467b23240ac89b0767491c6020b}

Valor inteiro (bps)

## Padrão {#section-dba669ea75354671bbdb3b520e2f6911}

Nenhum.
