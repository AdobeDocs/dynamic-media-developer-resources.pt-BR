# Cor{#color}

Tipo de dados de cor.

Valor da cor. Valor RGB hexadecimal e compactado, precedido opcionalmente por `0x`. Por exemplo, o valor RGB `128,255,0` pode ser especificado como `0x80ff00` ou `80ff00` .
