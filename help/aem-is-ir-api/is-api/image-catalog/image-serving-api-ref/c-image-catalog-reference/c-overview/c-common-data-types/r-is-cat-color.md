# Cor{#color}

Tipo de dados de cor.

Valor da cor. Valor de RGB hexadecimal compactado, opcionalmente precedido por `0x`. Por exemplo, o valor de RGB `128,255,0` pode ser especificado como `0x80ff00` ou `80ff00`.
