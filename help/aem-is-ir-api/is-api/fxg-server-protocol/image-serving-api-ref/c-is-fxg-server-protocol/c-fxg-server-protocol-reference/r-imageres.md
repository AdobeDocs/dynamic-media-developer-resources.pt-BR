# imageRes{#imageres}

Define a resolução de bitmaps incorporados em uma solicitação Web To Print.

` imageRes=< *`número`*>`

300 é recomendado ao imprimir. Quanto maior o número, mais tempo será necessário para renderizar a imagem.
