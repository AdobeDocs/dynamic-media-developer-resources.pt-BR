# imageRes{#imageres}

Define a resolução de bitmaps incorporados em uma solicitação para impressão da Web.

` imageRes=< *`número`*>`

300 é recomendado ao imprimir. Quanto maior o número, mais tempo será necessário para renderizar a imagem.
