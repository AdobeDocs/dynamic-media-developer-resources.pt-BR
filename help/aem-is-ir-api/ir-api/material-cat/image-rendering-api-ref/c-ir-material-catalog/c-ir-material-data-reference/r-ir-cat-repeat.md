# Repetir{#repeat}

Modo de repetição de textura. Especifica como as imagens de textura são colocadas lado a lado para preencher a superfície do público alvo.

## Propriedades {#section-cef4109cddf54ce095c3293d85bc412d}

Enum. Usado apenas para texturas repetíveis. Ignorado para todos os outros materiais.
