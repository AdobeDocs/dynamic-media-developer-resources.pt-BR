# Desinstalação no Windows{#uninstalling-on-windows}

Siga estas instruções para desinstalar a Renderização de imagem em um sistema Windows.

1. Inicie o desinstalador executando ` *[!DNL install_folder]*\_uninst\uninstaller.exe` ou, no Menu do Start, selecione **[!UICONTROL Programs]** > **[!UICONTROL Scene7]** > **[!UICONTROL Image Rendering]** > **[!UICONTROL Uninstall]**.
