Os modos de acolchoamento aleatório (14...18) podem ser usados para sintetizar texturas de imagens que não são prontamente repetíveis; o algoritmo cria texturas totalmente aleatórias ou parcialmente aleatórias com base na imagem original.

## Propriedades {#section-262bf540930d4890b678ea00cefe1909}

Atributo de material. Ignorado por materiais sólidos de cor, decalque e gabinete.

## Padrão {#section-e5bbd7d9fbb74852849e605d20f550bb}

`catalog::Repeat`, se o material for baseado em uma entrada de catálogo, caso contrário `0` (repetição direta).
